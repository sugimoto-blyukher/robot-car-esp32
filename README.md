# robot-car-esp32

##概要
サークル勧誘会で実演するESP32制御のロボットカー
PS3のコントローラーで操作できます

##必要部品
ESP32 dev kit c
PS3 dual sense 3
モータードライバー
ギアモーター2個
ゴムタイヤ四輪
5vバッテリー2個

##必須パッケージ
Arduino IDE
ESP32
Fork of PS3 Controller Host

##使い方
適当に車体を組み立ててESP32に書き込む
PS3 Dual Sense 3のMACアドレスをESP32のMACアドレスに合わせる
